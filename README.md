# 计算机图形学学习规划

### 📅 第一周：空间变换与渲染管线

**核心理论：** GAMES101 L1-L4（变换矩阵）
**核心实践：** OpenGL 环境搭建与 MVP 矩阵

| 天数          | 学习（理论）                     | 动手（编程）                                   | 考研复试准备（重点）                          |
| ------------- | -------------------------------- | ---------------------------------------------- | --------------------------------------------- |
| ==**Day 1**== | **GAMES101 L1-L2**: 线性代数回顾 | 配置 VS + GLFW + Glad + GLM 数学库。           | 了解为什么图形学偏爱矩阵运算。                |
| ==**Day 2**== | **LearnOpenGL**: Hello Triangle  | 编写第一个顶点/片段着色器，画出三角形。        | 理解 **VBO/VAO** 及其在内存中的位置。         |
| **Day 3**     | **GAMES101 L3**: 2D/3D 变换      | 实现物体的平移、旋转、缩放函数。               | 掌握**齐次坐标**：为什么 3D 坐标用 4 维向量？ |
| **Day 4**     | **GAMES101 L4**: 观测变换        | 编写 **MVP 矩阵**，将三角形变成 3D 立方体。    | 重点推导 **透视投影矩阵**。                   |
| **Day 5**     | **LearnOpenGL**: Camera          | 实现 WASD 漫游摄像机类，模拟在建筑中行走。     | 摄像机原理：LookAt 矩阵的构成。               |
| **Day 6**     | **GAMES101 L5**: 光栅化          | 了解 Z-Buffer 深度测试原理。                   | **Z-Buffer** 解决了什么问题？（遮挡关系）。   |
| **Day 7**     | 复习与调试                       | 整合代码，确保能流畅控制立方体在三维空间移动。 | 尝试解释渲染管线（Pipeline）全流程。          |

### 📅 第二周：着色模型与建筑数据

**核心理论：** GAMES101 L7-L9（着色、纹理）
**核心实践：** 冯氏光照、Assimp 模型加载

| 天数       | 学习（理论）                     | 动手（编程）                                          | 考研复试准备（重点）                             |
| ---------- | -------------------------------- | ----------------------------------------------------- | ------------------------------------------------ |
| **Day 8**  | **GAMES101 L7**: Shading 1       | 实现基础的环境光、漫反射。                            | **基础光照模型**：为什么法向量很重要？           |
| **Day 9**  | **GAMES101 L8**: Shading 2       | 实现镜面反射（Specular），完成 **Blinn-Phong** 模型。 | 比较不同着色频率（Flat vs. Gouraud vs. Phong）。 |
| **Day 10** | **GAMES101 L9**: Texture Mapping | 实现纹理贴图，给模型穿上“衣服”。                      | 什么是 **UV 坐标**？双线性插值的作用。           |
| **Day 11** | **模型解析理论**                 | 学习 **Assimp** 库，理解 3D 模型的文件结构。          | 建筑模型中“构件”是如何存储的。                   |
| **Day 12** | **动手实践**                     | 编写 Mesh 类和 Model 类，递归加载模型。               | 考察 C++ 的 **封装思想** 和 **递归算法**。       |
| **Day 13** | **BIM 数据导入**                 | 导出一个简单的建筑 OBJ 模型并用你的程序加载。         | 如何处理建筑模型中的**成千上万个三角面**？       |
| **Day 14** | 阶段性总结                       | 优化光照效果，让建筑模型看起来有体积感。              | 准备好回答：从磁盘加载模型到显卡显示的过程。     |

### 📅 第三周：BIM 专属特性开发

**核心理论：** GAMES101 L10-L12（几何表示）、着色器进阶
**核心实践：** 实时剖切、Alpha 混合、ImGui

| 天数       | 学习（理论）               | 动手（编程）                                           | 考研复试准备（重点）                       |
| ---------- | -------------------------- | ------------------------------------------------------ | ------------------------------------------ |
| **Day 15** | **GAMES101 L11**: 几何表示 | 了解离散几何与隐式几何的区别。                         | 建筑 BIM 是参数化表达，图形学是网格表达。  |
| **Day 16** | **Shader 进阶**            | 实现 **实时剖切 (Clipping Plane)**，通过坐标剔除像素。 | 核心函数：GLSL 中的 **discard**。          |
| **Day 17** | **深度与混合**             | 实现玻璃窗户的半透明效果。                             | 为什么渲染透明物体要**从远到近排序**？     |
| **Day 18** | **UI 交互集成**            | 集成 **Dear ImGui**，制作一个滑动条控制剖切面位置。    | 提升项目完成度，体现工程能力。             |
| **Day 19** | **性能优化初探**           | 学习**实例化渲染 (Instancing)**。                      | 若楼内有 1000 把相同的椅子，如何优化渲染？ |
| **Day 20** | 综合调试                   | 修复剖切时的“露馅”问题，调整漫游灵敏度。               | 考察排查 Bug 的思路。                      |
| **Day 21** | 视频录制                   | 录制一段带操作、带剖切演示的视频。                     | 留作复试面试时的辅助展示材料。             |
